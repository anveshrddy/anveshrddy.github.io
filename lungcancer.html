Lung Cancer Detection using Computer Vision

A CNN+LSTM pipeline for classifying malignant nodules in 3D CT scans, with Mask-RCNN for precise pixel-level localization — reducing diagnostic subjectivity for radiologists.

The American Cancer Society estimates approximately 238,340 new lung cancer cases in the US per year. Current diagnosis requires expert radiologists to manually inspect CT scan sequences — a process that is inherently subjective and time-consuming. We set out to build a deep learning model that could provide an objective, reproducible assessment: is this nodule malignant, and exactly where is it?

Methodology

CT scans are inherently 3D volumetric data. Rather than processing the full 3D volume (computationally expensive and architecturally complex), we treated each scan as a sequence of 2D axial slices. This enabled us to apply a two-stage pipeline:

Stage 1 — CNN: A convolutional neural network processes each 2D slice independently, extracting spatial features relevant to malignancy detection.

Stage 2 — LSTM: The feature vectors from each slice are fed sequentially into a Long Short-Term Memory network, which learns inter-slice relationships — capturing how a nodule evolves across the depth of the scan.

Prototype Validation

Before training on medical data, we validated the CNN+LSTM architecture on a synthetic dataset. We modeled 3D polyhedra (20, 21, and 22-faced shapes) in SolidWorks, sliced them into 2D images, and trained the model to distinguish between them individually and in composite scenes. This achieved ~70% accuracy, confirming the architectural approach was sound.

Main Model

The main model was trained on CT scans from 500 distinct patients with confirmed malignant nodules. Using Cross-Entropy loss and Adam optimizer, we achieved approximately 65% classification accuracy — meaningful for the task but insufficient for clinical deployment.

Localization with Mask-RCNN

Classification alone (malignant / benign) isn't enough for clinical use — radiologists need to know where the nodule is. We added Mask-RCNN as a second stage: when the classification model predicted malignancy above a confidence threshold, Mask-RCNN performed instance segmentation to delineate the exact nodule boundary.

Mask-RCNN's per-pixel prediction provides exact contour information rather than bounding boxes — critical for radiologists estimating nodule size and morphology when making treatment decisions.

Limitations & Lessons

The primary limitation was data imbalance: we had 500 patients with malignant nodules but insufficient benign examples. Without balanced negative examples, the model struggled to learn the boundaries between healthy and malignant tissue. Augmentation and synthetic negative generation would be the first improvements in a follow-up.

Despite the accuracy ceiling, the project demonstrated a complete end-to-end pipeline from volumetric medical data to pixel-level localization — and the architectural pattern of CNN spatial features + LSTM temporal reasoning applies broadly to any sequential 3D volume analysis problem.
