Bag of Visual Words for Scene Classification

An end-to-end scene classification system using the Bag of Visual Words representation and Spatial Pyramid Matching — classifying 8 scene categories from the SUN image dataset.

Scene classification is a foundational task in computer vision. Given an image, the system must determine which of a set of predefined scene categories it belongs to. This project built a complete pipeline — from raw pixels to classification — using the Bag of Visual Words (BoW) approach, a precursor to modern deep learning that still offers strong intuitions about feature representation.

Visual Dictionary Construction

The first step is building a "visual dictionary" — analogous to a word dictionary in NLP. We start by applying a Harris corner detector to find interest points in training images, then extract local patch descriptors around each interest point using a filter bank.

The filter responses are collected across all training images, then clustered with k-means to produce a dictionary of K visual words. Each cluster center becomes a "word" — a representative local appearance pattern that occurs frequently in the dataset.

Image Representation with Spatial Pyramid Matching

Once we have a dictionary, each image is represented as a histogram of visual word occurrences. But a flat histogram loses all spatial information. We address this with Spatial Pyramid Matching (SPM): the image is divided into grids at multiple resolutions (1×1, 2×2, 4×4), and histograms are computed for each cell. Concatenating these produces a spatially-aware feature vector that captures where in the image different visual patterns appear.

Classification Results

Using an SVM classifier on the SPM features, we evaluated classification accuracy on 8 scene categories from the SUN dataset. The visual word histograms are clearly discriminative: two aquarium images share a very similar histogram, while an aquarium and laundromat have distinctly different distributions.

Why This Matters

BoW approaches have been largely superseded by CNNs for large-scale recognition. But understanding them is valuable — they illuminate how deep networks work: convolutional filters are essentially learned generalizations of the hand-crafted filter banks used here, and pooling operations are a smooth version of the histogram computation. The spatial pyramid is a direct precursor to spatial pooling in modern architectures.

For 8 categories with limited data, BoW + SPM remains surprisingly competitive and trains orders of magnitude faster than a neural network.
