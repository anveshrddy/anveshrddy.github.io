Fashion is All You Need

Texture-controlled and pose-controlled image generation for fashion design — using Stable Diffusion with cloth segmentation and inpainting to synthesize new clothing on reference human models.

The intersection of AI and fashion design offers real economic and environmental value: generating synthetic clothing images eliminates the need for physical prototypes, reduces waste from unsold inventory, and lowers the cost of fashion photography. The challenge is giving designers sufficient control over the generated output — particularly cloth texture, which can't be adequately described in text.

Prior Work

We surveyed several relevant methods: Repaint for texture transfer, DiffFashion for garment generation, InstructPix2Pix for text-guided image editing, and Stable Diffusion as the generative backbone. While these methods produce high-quality images, none offered fine-grained control over cloth texture when applied to fashion — text prompts alone couldn't capture specific fabric patterns or materials.

Our Approach

Our insight: cloth texture is visual information, not textual. So instead of describing it, we show it. The pipeline takes three inputs — a reference image of the person, a reference texture image, and a text prompt — and generates a new outfit that applies the given texture.

The pipeline proceeds in three steps:

Step 1 — Cloth Segmentation: We use a segmentation model to isolate the clothing region from the reference image. This mask defines the regeneration region and preserves the background and body.

Step 2 — Texture Augmentation: The segmentation mask is augmented with the target texture image, creating an inpainting-ready input that visually shows the model where and what texture to apply.

Step 3 — Stable Diffusion Inpainting: We run the augmented image through Stable Diffusion's inpainting pipeline with the prompt "Complete woman's dress in high quality." The model fills in the masked region with the texture pattern while maintaining pose consistency.

Results

The method produced visually convincing results across multiple texture types (denim, floral, solid colors) and body poses. The segmentation mask effectively guided the inpainting to the target region without bleeding into the background or skin.

Takeaways

This project demonstrated that combining explicit spatial control (segmentation masks) with visual texture conditioning gives substantially better results than text-only prompting for fashion generation. The main limitation is that the Stable Diffusion model sometimes interprets the augmented texture mask as background rather than instruction — a challenge addressable with ControlNet conditioning or fine-tuned inpainting models.
